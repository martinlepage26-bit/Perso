# Route Map Template

## Workflow Anchor
Map one specific decision/action under scrutiny.

## Route Nodes
1. Input source
2. Preprocess/transform layer
3. Model/rules layer
4. Retrieval/tool layer
5. Human review gate
6. Output destination
7. Logging layer
8. Override path
9. Incident/shutdown path

## Control Bindings
For each route node, reference:
- threshold IDs
- decision rights role(s)
- evidence artifact IDs
