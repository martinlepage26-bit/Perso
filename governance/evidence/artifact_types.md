# Evidence Artifact Types

- `model_output_log`: route outputs and metadata under case linkage.
- `legal_review_record`: legal adjudication for gated outputs.
- `governance_release_approval`: formal approval at governance gate.
- `provenance_snapshot`: immutable metadata linking case, artifacts, and hashes.
- `provenance_drift_ticket`: remediation record when provenance mismatch occurs.
- `claim_language_scan`: scan report of high-risk policy/legal language.
- `destination_classification_record`: release destination classification proof.
- `reviewer_decision_log`: human review outcomes and rationale.
- `model_change_record`: model/version/provider changes.
- `vendor_change_record`: contract/vendor control changes.
